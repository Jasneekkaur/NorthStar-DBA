# NorthStar Urban Mobility and Logistics
## Databases and Analytics (DBA) — Coursework Assignment

This repository contains the full analytical solution for the NorthStar 
Urban Mobility and Logistics case study, completed as part of the 
Databases and Analytics module.

## Repository Contents

| File | Description |
|---|---|
| NorthStar_SQL_R_Analytics.ipynb | SQL in R and R Analytics notebook |
| NorthStar_Python_MongoDB.ipynb | Python processing, MongoDB and Query Optimisation notebook |
| customers.csv | Customer master data |
| orders.csv | Service orders data |
| deliveries.csv | Delivery outcomes data |
| drivers.csv | Driver workforce data |
| vehicles.csv | Fleet asset data |
| hubs.csv | Operational hubs data |
| incidents.csv | Incident records |
| complaints.csv | Customer complaints data |
| app_events.csv | Mobile platform event data |

## Tools and Technologies Used
- R (sqldf, ggplot2, dplyr)
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- MongoDB Atlas (PyMongo)
- Google Colab

## Key Analysis Areas
- Delivery failure and delay patterns
- Hub and zone performance analysis
- Driver behaviour and override analysis
- Vehicle maintenance impact
- Customer complaint trends
- NoSQL database design for complex operational data
- Query optimisation using indexes and explain plans
